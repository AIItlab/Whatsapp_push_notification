How to Update

Regularly update the installed WAHA to keep your bot stable, usable, and secure.

  • Core
  • Plus

On this page

  • Core
  • Plus
💡
Please test all new versions in the development environment before updating production!

Core #

For Core version the command is

docker pull devlikeapro/whatsapp-http-api

Plus #

For Plus version, we use login to get the image before:

docker login -u devlikeapro -p {PASSWORD}
docker pull devlikeapro/whatsapp-http-api-plus
docker logout

Read more about how to get PASSWORD for Plus Version →
